# Notification Sounds Setup

This directory contains notification sounds from Pixabay (free to use).

## Download these sounds from Pixabay:

1. **Shamisen** - Japanese string instrument
   - Search: "shamisen" or "japanese instrument"
   - Duration: ~1 second

2. **Arcade** - Retro game sound
   - Search: "arcade" or "retro game"
   - Duration: ~3 seconds

3. **Ping** - Clean notification
   - Search: "notification ping" or "bell"
   - Duration: ~1 second

4. **Quick Ping** - Short alert
   - Search: "short ping" or "alert"
   - Duration: ~1 second

5. **Success/Magic** - Positive sound
   - Search: "success" or "magic chime"
   - Duration: ~2 seconds

6. **Complete** - Task done
   - Search: "complete" or "achievement"
   - Duration: ~3 seconds

7. **Celebration** - Cheerful sound
   - Search: "celebration" or "happy"
   - Duration: ~3 seconds

8. **Long Sound** - Extended notification
   - Search: "notification long"
   - Duration: ~5+ seconds

9. **Attention** - Urgent alert
   - Search: "attention" or "warning"
   - Duration: ~3 seconds

10. **Cheerful** - Fun sound
    - Search: "cheerful" or "upbeat"
    - Duration: ~3 seconds

## File Naming:
Download as MP3 and rename to:
